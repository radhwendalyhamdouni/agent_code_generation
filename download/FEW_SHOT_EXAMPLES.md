# أمثلة التدريب - Few-Shot Examples

هذا الملف يحتوي على مجموعة من الأمثلة البرمجية في لغة المرجع لاستخدامها في تدريب نماذج الذكاء الاصطناعي. كل مثال يوضح "المهمة المطلوبة" و"الكود المنفذ".

## مثال 1: طباعة رسالة ترحيب

**المهمة:** اكتب برنامجاً يطبع رسالة ترحيب بالمستخدم.

**الكود:**
```almarjaa
# برنامج ترحيب بسيط
اطبع("مرحباً بك في لغة المرجع!")؛
اطبع("هذه لغة برمجة عربية بالكامل")؛
```

**المخرجات:**
```
مرحباً بك في لغة المرجع!
هذه لغة برمجة عربية بالكامل
```

## مثال 2: تعريف المتغيرات والثوابت

**المهمة:** عرّف متغيراً للاسم بقيمة "أحمد" وثابتاً للـ pi بقيمة 3.14، ثم اطبعها.

**الكود:**
```almarjaa
# تعريف متغيرات
متغير الاسم = "أحمد"؛
متغير العمر = ٢٥؛
متغير الطول = ١٧٥.٥؛

# تعريف ثابت
ثابت pi = ٣.١٤١٥٩؛

اطبع("الاسم: " + الاسم)؛
اطبع("العمر: " + نص(العمر))؛

# استخدام الثابت
متغير مساحة = pi * ٥ * ٥؛
اطبع("مساحة الدائرة: " + نص(مساحة))؛
```

**المخرجات:**
```
الاسم: أحمد
العمر: ٢٥
مساحة الدائرة: ٧٨.٥٣٩٧٥
```

**الكود:**

## مثال 4: العمليات الحسابية

**المهمة:** اكتب دالة حاسبة تقوم بالجمع والطرح والضرب والقسمة.

**الكود:**
```almarjaa
دالة جمع(أ، ب) { أرجع أ + ب؛ }
دالة طرح(أ، ب) { أرجع أ - ب؛ }
دالة ضرب(أ، ب) { أرجع أ * ب؛ }
دالة قسمة(أ، ب) {
    إذا ب == ٠ {
        أرجع "خطأ: لا يمكن القسمة على صفر"؛
    }
    أرجع أ / ب؛
}

اطبع("جمع ١٠ + ٥ = " + نص(جمع(١٠، ٥)))؛
اطبع("طرح ١٠ - ٥ = " + نص(طرح(١٠، ٥)))؛
اطبع("ضرب ١٠ × ٥ = " + نص(ضرب(١٠، ٥)))؛
اطبع("قسمة ١٠ ÷ ٥ = " + نص(قسمة(١٠، ٥)))؛
```

**المخرجات:**
```
جمع ١٠ + ٥ = ١٥
طرح ١٠ - ٥ = ٥
ضرب ١٠ × ٥ = ٥٠
قسمة ١٠ ÷ ٥ = ٢
```

```almarjaa
# مفاتيح وقيم
اطبع("المفاتيح: " + نص(مفاتيح(طالب)))؛
اطبع("القيم: " + نص(قيم(طالب)))؛
```

**المخرجات:**
```
الاسم: أحمد
العمر: ٢٠
```

## مثال 15: معالجة الأخطاء

## مثال 18: الزخارف (Decorators)

```almarjaa
متجر نتيجة = حساب_مضروب(٥)؛
اطبع("مضروب ٥ = " + نص(نتيجة))؛
```

**المخرجات:**
```

انتهى التنفيذ!
```

## مثال 20: برنامج شامل - نظام إدارة الطلاب

**المهمة:** اكتب برنامجاً شاملاً لإدارة بيانات الطلاب.

**الكود:**
```almarjaa
# تعريف فئة بيانات للطالب
بيانات طالب {
    الاسم: نص،
    العمر: رقم،
    الدرجات: [رقم]
}

# دالة حساب المعدل
دالة حساب_المعدل(الطالب) {
    متغير مجموع_الدرجات = مجموع(الطالب.الدرجات)؛
    أرجع مجموع_الدرجات / طول(الطالب.الدرجات)؛
}

# دالة تحديد التقدير
دالة تحديد_التقدير(المعدل) {
    إذا المعدل >= ٩٠ { أرجع "ممتاز"؛ }
    وإذا المعدل >= ٨٠ { أرجع "جيد جداً"؛ }
    وإذا المعدل >= ٧٠ { أرجع "جيد"؛ }
    وإذا المعدل >= ٦٠ { أرجع "مقبول"؛ }
    وإلا { أرجع "راسب"؛ }
}

# إنشاء قائمة طلاب
متغير طلاب = [
    طالب {الاسم: "أحمد"، العمر: ٢٠، الدرجات: [٨٥، ٩٠، ٧٨، ٩٢]}،
```

## إرشادات الاستخدام

### للتدريب على نموذج AI:

1. **أدخل المهمة** كـ User Prompt
2. **الكود المتوقع** كـ Assistant Response
3. **المخرجات** كمرجع للتحقق

### صيغة الـ Prompt المقترحة:
```
المهمة: [وصف المهمة]
اللغة: لغة المرجع (Al-Marjaa)
المطلوب: كتابة كود يحقق المهمة المحددة

[الكود المتوقع]
```

### أمثلة على الـ Prompts:

**Prompt بسيط:**
```
اكتب برنامجاً في لغة المرجع يطبع "مرحباً بالعالم"
```

**Prompt متوسط:**
```
اكتب دالة في لغة المرجع تحسب مضروب رقم معين باستخدام التكرار
```

**Prompt متقدم:**
```
اكتب نظاماً في لغة المرجع لإدارة مكتبة يدعم إضافة الكتب والبحث عنها والإعارة
```
