# مواصفات لغة المرجع (Al-Marjaa Language Specification)

**الإصدار:** 3.2.0
**تاريخ الإصدار:** 2026-02
**الحالة:** مستقر

## 1. مقدمة

لغة المرجع هي لغة برمجة عربية حديثة مصممة لتكون سهلة التعلم والاستخدام للمتحدثين باللغة العربية. تتميز بكلمات مفتاحية عربية بالكامل، مع دعم كامل للأرقام العربية والنصوص ثنائية الاتجاه. بنيت اللغة باستخدام Rust مما يوفر أداءً عالياً وأماناً في الذاكرة. تجمع اللغة بين بساطة Python وقوة أنظمة الأنواع الحديثة، مع دعم مدمج للذكاء الاصطناعي وواجهات المستخدم.

### 1.1 الفلسفة التصميمية

تعتمد لغة المرجع على عدة مبادئ أساسية تجعلها فريدة ومتميزة عن غيرها من لغات البرمجة. أولاً، الأسبقية للعربية حيث تكون جميع الكلمات المفتاحية والرسائل والأخطاء باللغة العربية، مما يقلل الحواجز اللغوية للمبتدئين العرب. ثانياً، البساطة والوضوح حيث يكون الكود قابلاً للقراءة بسهولة ويشبه اللغة العربية الطبيعية في تركيبه. ثالثاً، الأمان والقوة حيث توفر اللغة نظام أنواع قوياً مع رسائل خطأ واضحة ومفيدة بالعربية. رابعاً، التوسعية حيث تدعم اللغة ميزات متقدمة مثل البرمجة الشيئية والبرمجة الوظيفية والذكاء الاصطناعي.

### 1.2 مميزات اللغة الرئيسية

توفر لغة المرجع مجموعة واسعة من الميزات التي تجعلها مناسبة لمختلف أنواع المشاريع. تشمل هذه الميزات الكلمات المفتاحية العربية بالكامل مع دعم الأشكال المختلفة مثل "إذا" و"اذا"، الدعم الكامل للأرقام العربية الشرقية من ٠ إلى ٩، أنواع بيانات غنية تشمل الأرقام والنصوص والمنطقية والقوائم والقواميس، دوال مع دعم القيم الافتراضية والتعليقات التوضيحية للأنواع، ميزات متقدمة على نمط Python مثل القوائم الاستيعابية والتفكيك والمولدات، دعم البرمجة الشيئية بالأصناف والوراثة، دعم فئات البيانات والتعدادات، دعم الذكاء الاصطناعي مع Tensors والشبكات العصبية، نظام وحدات قوي لتنظيم الكود، ومعالجة أخطاء متقدمة مع حاول/امسك.

## 2. البنية المعجمية (Lexical Structure)

### 2.1 المحارف المدعومة

تدعم لغة المرجع مجموعة واسعة من المحارف لتسهيل الكتابة باللغة العربية. تشمل الحروف العربية الأساسية جميع حروف الأبجدية العربية من "ا" إلى "ي" مع الأشكال الشائعة مثل التاء المربوطة "ة" والهمزة بأنواعها "ؤ، إ، أ، آ". بالإضافة إلى ذلك، تدعم اللغة الحروف اللاتينية الصغيرة والكبيرة، والأرقام العربية الشرقية من ٠ إلى ٩ والأرقام اللاتينية من 0 إلى 9. كما يمكن استخدام الشرطة السفلية "_" والشرطة الطويلة "ـ" داخل المعرفات لتحسين القراءة.

### 2.2 التعليقات

توفر لغة المرجع عدة أنواع من التعليقات لتوثيق الكود بشكل فعال. التعليقات أحادية السطر تبدأ بالرمز # وتستمر حتى نهاية السطر، وهي مفيدة للتعليقات القصيرة. التعليقات أحادية السطر البديلة تبدأ بـ // وتستمر حتى نهاية السطر أيضاً. أما التعليقات متعددة الأسطر فتبدأ بـ /* وتنتهي بـ */ ويمكن أن تمتد عبر عدة أسطر، وهي مفيدة للتعليقات الطويلة وتوثيق الدوال والوحدات البرمجية.

```almarjaa
# هذا تعليق أحادي السطر

// هذا أيضاً تعليق أحادي السطر (نمط C)

/* هذا تعليق
   متعدد الأسطر
   يمكن أن يمتد لعدة أسطر */
```

### 2.3 المعرفات (Identifiers)

المعرفات في لغة المرجع تتبع قواعد محددة لضمان القراءة والوضوح. يجب أن يبدأ المعرف بحرف عربي أو لاتيني أو شرطة سفلية، ويمكن أن يحتوي على أحرف وأرقام وشرطات سفلية. من الممكن أيضاً استخدام الشرطة الطويلة العربية "ـ" داخل المعرفات لتحسين القراءة. لا يمكن استخدام الكلمات المحجوزة كمعرفات إلا في سياقات محددة. المعرفات حساسة لحالة الأحرف في الحروف اللاتينية، مما يعني أن "اسم" و"الاسم" و"الإسم" كلها معرفات مختلفة.

```almarjaa
متغير اسم_الطالب = "أحمد"؛     # معرف صالح
متغير العمر = ٢٥؛              # معرف صالح
متغير total_count = ١٠٠؛       # معرف صالح (لاتيني)
متغير رقم_الهاتف = "05xxxxxxxx"؛ # معرف صالح
```

### 2.4 الكلمات المحجوزة (Keywords)

تنقسم الكلمات المحجوزة في لغة المرجع إلى عدة فئات حسب وظيفتها:

#### فئة الدوال والتحكم في التدفق
تشمل هذه الفئة الكلمات المستخدمة لتعريف الدوال والتحكم في سير البرنامج. الكلمة "دالة" تستخدم لتعريف الدوال، و"fn" هي اختصار بديل. "أرجع" و"ارجع" تستخدم لإرجاع قيمة من الدالة. "إذا" و"اذا" للشرط، "وإذا" و"واذا" للشرط المتعدد، "وإلا" و"والا" و"غير_ذلك" للفرع الآخر. "طالما" للحلقة الشرطية، "لكل" لحلقة التكرار على العناصر، "في" للدلالة على العنصر في المجموعة. "توقف" لكسر الحلقة، "أكمل" و"اكمل" للانتقال للتكرار التالي. "كرر" للتكرار عدد محدد من المرات، "مرة" و"مرات" لتحديد عدد التكرارات.

#### فئة المتغيرات والثوابت
تتضمن هذه الفئة الكلمات المستخدمة للإعلان عن المتغيرات والثوابت. "متغير" للإعلان عن متغير قابل للتعديل، و"م" اختصار لها. "ثابت" للإعلان عن ثابت لا يمكن تعديله، و"ث" اختصار لها.

#### فئة القيم المنطقية والخالية
تشمل هذه الفئة القيم الأساسية في اللغة. "صح" و"true" للقيمة المنطقية الصحيحة، "خطأ" و"خطا" و"false" للقيمة المنطقية الخاطئة، "لا_شيء" و"لاشيء" و"null" للقيمة الفارغة.

#### فئة البرمجة الشيئية
تتضمن هذه الفئة الكلمات المتعلقة بالبرمجة كائنية التوجه. "صنف" لتعريف صنف جديد، "هذا" للإشارة إلى الكائن الحالي، "جديد" لإنشاء كائن جديد، "أصل" و"اصل" للإشارة إلى الصنف الأب.

#### فئة الوحدات والاستيراد
تشمل كلمات إدارة الوحدات. "استورد" لاستيراد وحدة أو دالة، "صدر" لتصدير دالة أو وحدة، "وحدة" لتعريف وحدة، "استخدم" لاستخدام وحدة، "من" لتحديد مصدر الاستيراد، "بوصف" لتسمية الاستيراد باسم مستعار، "كـ" و"ك" بديل لـ "بوصف".

#### فئة معالجة الأخطاء
تتضمن كلمات التعامل مع الاستثناءات. "حاول" لبدء كتلة معالجة الأخطاء، "امسك" للقبض على الخطأ، "أخيراً" و"اخيراً" و"أخيرا" لكتلة التنفيذ النهائية، "ألقِ" و"الق" لرمي خطأ.

#### فئة العمليات المنطقية
تشمل الكلمات المستخدمة في العمليات المنطقية. "و" للعملية المنطقية AND، "أو" و"او" للعملية المنطقية OR، "ليس" للنفي المنطقي.

#### فئة الدوال المدمجة
تتضمن الدوال الأساسية المدمجة في اللغة. "اطبع" و"طبع" و"أطبع" للطباعة، "ادخل" و"دخل" للإدخال، "نوع" للحصول على نوع القيمة، "طول" للحصول على طول القائمة أو النص.

#### فئة التعدادات وفئات البيانات
تشمل الكلمات المتعلقة بتعريف هياكل البيانات. "تعداد" لتعريف تعداد، "حالة" للحالة داخل التعداد أو المطابقة، "افتراضي" للحالة الافتراضية في المطابقة، "بيانات" لتعريف فئة بيانات.

#### فئة الميزات المتقدمة
تتضمن كلمات الميزات المتقدمة. "لامدا" و"دالة_صغيرة" للدوال المجهولة، "غير_متزامن" للبرمجة غير المتزامنة، "انتظر" للانتظار حتى اكتمال عملية غير متزامنة، "أعطِ" و"اعط" للمولدات، "مع" لمديري السياق.

#### فئة واجهات المستخدم
تشمل الكلمات المستخدمة لبناء واجهات المستخدم. للتخطيط نجد "صف" و"صف_أفقي" و"عمود" و"صف_عمودي" و"شبكة_تخطيط" و"شبكة_عرض" و"مرن" و"كومة" و"التفاف" و"فجوة" و"محاذاة" و"تبرير" و"حشو" و"هامش". للمكونات نجد "زر" و"نص_معروض" و"عرض_نص" و"إدخال" و"ادخال" و"اختيار" و"خانة" و"راديو" و"منزلق" و"تقدم" و"مؤقت" و"بطاقة" و"قائمة" و"جدول" و"استمارة" و"نموذج_إدخال" و"تسمية" و"صورة" و"أيقونة" و"ايقونة" و"شارة" و"صورة_شخصية" و"تلميح" و"نافذة" و"نافذة_منبثقة" و"تنبيه" و"منبثق". للثيمات نجد "لون" و"خط" و"حجم" و"عرض" و"ارتفاع" و"حدود" و"ظل" و"خلفية". للرسوم البيانية نجد "رسم" و"رسم_بياني" و"رسم_خطي" و"رسم_أعمدة" و"رسم_دائري" و"رسم_مساحي". للرسوم المتحركة نجد "حرك" و"حركة" و"انتقال" و"مدة" و"تأخير" و"تخفيف". للأحداث نجد "نقر" و"تغيير" و"إرسال" و"ارسال" و"تركيز" و"ضبابية" و"تحويم" و"تمرير". للربط نجد "ربط" و"راقب" و"محسوب" و"راقب_التغييرات".

#### فئة الذكاء الاصطناعي
تتضمن الكلمات المستخدمة في تطبيقات الذكاء الاصطناعي. "أونكس" و"onnx" للتعامل مع نموذج ONNX، "نموذج" لتعريف نموذج، "حمّل" و"حمل" لتحميل نموذج، "احفظ" و"حفظ" لحفظ نموذج، "استدل" و"استدلال" للاستدلال، "موتر" للتعامل مع الموترات، "شكل" لتحديد شكل الموتر، "مخرج" للمخرجات، "طبقة" لتعريف طبقة، "كثيف" و"خطي" للطبقة الكثيفة، "التفاف" و"conv" للطبقة الالتفافية، "تجميع" و"pool" للتجميع، "طبع" و"تسوية_قيم" للتطبيع، "إسقاط" و"اسقاط" للـ Dropout، "تسوية" و"flatten" للتسوية، "إعادة_تشكيل" و"reshape" لإعادة التشكيل، "تنشيط" للدوال التنشيطية، "سوفت_ماكس" و"softmax" للـ Softmax، "ريلو" و"relu" للـ ReLU، "سيجمويد" و"sigmoid" للـ Sigmoid، "دفعة" و"batch" للدفعة، "درّب" و"درب" و"تدريب" للتدريب، "توقع" و"تنبؤ" للتنبؤ، "محسّن" و"محقق" للمحسن، "خسارة" لدالة الخسارة.

## 3. أنواع البيانات (Data Types)

### 3.1 الأنواع الأساسية

#### الرقم (Number)
تدعم لغة المرجع الأرقام العشرية (floating-point) بدقة 64 بت. يمكن كتابة الأرقام بالأرقام العربية الشرقية أو اللاتينية. تدعم اللغة أيضاً الأرقام السالبة والأرقام العشرية والأرقام السداسية عشر والثنائية. القيم الخاصة المدعومة تشمل "ليس_رقم" لـ NaN، "لانهاية" لـ Infinity، و"ناقص_لانهاية" لـ -Infinity.

```almarjaa
متغير عدد_صحيح = ٤٢؛           # رقم صحيح
متغير عدد_عشري = ٣.١٤١٥٩؛     # رقم عشري
متغير رقم_سالب = -١٠٠؛        # رقم سالب
متغير رقم_سداسي = 0xFF؛        # رقم سداسي عشر (255)
متغير رقم_ثنائي = 0b1010؛      # رقم ثنائي (10)
```

#### النص (String)
تمثل النصوص سلاسل من المحارف Unicode. يمكن تعريفها بعلامات اقتباس مفردة أو مزدوجة أو باستخدام الفاصلة العكسية للنصوص متعددة الأسطر. تدعم النصوص تسلسلات الهروب مثل \n للسطر الجديد و\t للجدولة و\\ للشرطة العكسية و\" لعلامة الاقتباس.

```almarjaa
متغير نص_بسيط = "مرحباً بالعالم"؛
متجر نص_مفرد = 'نص بعلامة مفردة'؛
متغير نص_متعدد = `هذا نص
يمتد عبر
عدة أسطر`؛
```

#### المنطقي (Boolean)
نوع المنطقي له قيمتان فقط: "صح" للقيمة الصحيحة و"خطأ" للقيمة الخاطئة. يمكن استخدام "true" و"false" كبديل لاتيني.

```almarjaa
متغير نشط = صح؛
متغير معطل = خطأ؛
متغير using_english = true؛
```

#### الفراغ (Null)
يمثل غياب القيمة باستخدام "لا_شيء" أو "null".

```almarjaa
متغير قيمة_فارغة = لا_شيء؛
متغير empty_value = null؛
```

### 3.2 أنواع المجموعات

#### القائمة (List)
القوائم هي مجموعات مرتبة من القيم يمكن أن تكون من أنواع مختلفة. يتم الوصول للعناصر بالفهرس الذي يبدأ من صفر. القوائم قابلة للتعديل ويمكن تغيير حجمها.

```almarjaa
متغير أرقام = [١، ٢، ٣، ٤، ٥]؛
متغير أسماء = ["أحمد"، "محمد"، "علي"]؛
متجر مختلط = [١، "نص"، صح، لا_شيء]؛

# الوصول للعناصر
اطبع(أرقام[٠])؛              # ١
اطبع(أرقام[طول(أرقام) - ١])؛ # ٥
```

#### القاموس (Dictionary)
القواميس هي مجموعات من أزواج المفتاح-القيمة. يمكن استخدام أي قيمة كمفتاح. توفر وصولاً سريعاً للقيم بواسطة المفاتيح.

```almarjaa
متغير طالب = {
    الاسم: "أحمد"،
    العمر: ٢٠،
    التخصص: "علوم الحاسب"
}؛

# الوصول للقيم
اطبع(طالب.الاسم)؛        # أحمد
اطبع(طالب["العمر"])؛     # ٢٠
```

### 3.3 أنواع الدوال

#### الدالة العادية (Function)
الدوال هي كتل من الكود قابلة لإعادة الاستخدام. يمكن أن تأخذ معاملات وترجع قيماً. يمكن أن يكون للمعاملات قيم افتراضية وتعليقات توضيحية للنوع.

```almarjaa
دالة جمع(أ، ب) {
    أرجع أ + ب؛
}

دالة ترحيب(الاسم = "ضيف") {
    أرجع "مرحباً " + الاسم + "!"؛
}

دالة مساحة(الطول: رقم، العرض: رقم) -> رقم {
    أرجع الطول * العرض؛
}
```

#### الدالة المجهولة (Lambda)
الدوال المجهولة هي دوال صغيرة بدون اسم، تُعرّف باستخدام الصيغة (معاملات) => تعبير.

```almarjaa
متغير مربع = (ن) => ن ^ ٢؛
متغير مجموع = (أ، ب) => أ + ب؛
```

### 3.4 أنواع متقدمة

#### الصنف (Class)
الأصناف هي قوالب لإنشاء الكائنات. تحتوي على حقول ودوال.

```almarjaa
صنف شخص {
    متغير الاسم؛
    متغير العمر؛
    
    دالة شخص(الاسم، العمر) {
        هذا.الاسم = الاسم؛
        هذا.العمر = العمر؛
    }
    
    دالة تعريف() {
        أرجع "أنا " + هذا.الاسم + " وعمري " + نص(هذا.العمر)؛
    }
}

متغير أحمد = جديد شخص("أحمد"، ٢٥)؛
```

#### التعداد (Enum)
التعدادات تُعرّف مجموعة من القيم الثابتة المسماة.

```almarjaa
تعداد لون {
    أحمر،
    أخضر،
    أزرق
}

تعداد حالة {
    نشط = ١،
    معطل = ٠،
    معلق = ٢
}

اطبع(لون.أحمر)؛       # أحمر
اطبع(حالة.نشط)؛       # ١
```

#### فئة البيانات (Data Class)
فئات البيانات هي أصناف مبسطة لتخزين البيانات.

```almarjaa
بيانات منتج {
    اسم: نص،
    سعر: رقم،
    كمية = ١
}

متغير كتاب = منتج {اسم: "كتاب برمجة"، سعر: ٥٠، كمية: ٣}؛
اطبع(كتاب.اسم)؛       # كتاب برمجة
```

#### الموتر (Tensor)
الموترات هي مصفوفات متعددة الأبعاد للاستخدام في تطبيقات الذكاء الاصطناعي.

```almarjaa
متجر مصفوفة = موتر([١، ٢، ٣، ٤]، [٢، ٢])؛  # مصفوفة 2x2
```

## 4. المشغلات (Operators)

### 4.1 المشغلات الحسابية

تدعم لغة المرجع المشغلات الحسابية الأساسية بترتيب أولوية محدد. المشغل + للجمع، - للطرح، * للضرب، / للقسمة، % لباقي القسمة، ^ للأس، // للقسمة الصحيحة.

```almarjaa
متغير مجموع = ١٠ + ٥؛        # ١٥
متغير فرق = ١٠ - ٣؛          # ٧
متجر حاصل = ٤ * ٣؛           # ١٢
متجر قسمة = ١٠ / ٢؛          # ٥
متجر باقي = ١٠ % ٣؛          # ١
متجر أس = ٢ ^ ٣؛             # ٨
متجر قسمة_صحيحة = ١٠ // ٣؛   # ٣
```

### 4.2 مشغلات المقارنة

تُستخدم لمقارنة قيمتين وتُرجع قيمة منطقية. == للمساواة، != لعدم المساواة، < لأصغر من، > لأكبر من، <= لأصغر أو يساوي، >= لأكبر أو يساوي.

```almarjaa
إذا س == ١٠ { ... }
إذا س != ٥ { ... }
إذا س < ١٠ { ... }
إذا س > ٠ { ... }
إذا س <= ١٠ { ... }
إذا س >= ٠ { ... }
```

### 4.3 المشغلات المنطقية

تُستخدم للعمليات المنطقية. "و" للعملية AND، "أو" للعملية OR، "ليس" للنفي.

```almarjaa
إذا س > ٠ و ص > ٠ { ... }
إذا س < ٠ أو ص < ٠ { ... }
إذا ليس(س == ٠) { ... }
```

### 4.4 مشغلات الإسناد

تُستخدم لتعيين قيم للمتغيرات. = للإسناد البسيط، += للإسناد مع الجمع، -= للإسناد مع الطرح، *= للإسناد مع الضرب، /= للإسناد مع القسمة، %= للإسناد مع الباقي، ^= للإسناد مع الأس.

```almarjaa
س = ١٠؛
س += ٥؛      # س = ١٥
س -= ٣؛      # س = ١٢
س *= ٢؛      # س = ٢٤
س /= ٤؛      # س = ٦
```

### 4.5 المشغلات المتقدمة

#### مشغل الدمج الفارغ (Null Coalescing)
يُرجع القيمة الأولى إذا لم تكن لا_شيء، وإلا يُرجع القيمة الثانية.

```almarjaa
متغير أ = لا_شيء؛
متغير ب = "قيمة افتراضية"؛
متجر نتيجة = أ ؟؟ ب؛    # "قيمة افتراضية"
```

#### مشغل الأنابيب (Pipe Operator)
يُمرر القيمة كمعامل أول للدالة.

```almarjaa
دالة مضاعف(س) {
    أرجع س * ٢؛
}
متغير نتيجة = ٥ |> مضاعف؛    # ١٠
```

#### الوصول الآمن (Optional Chaining)
يُرجع لا_شيء إذا كان الكائن لا_شيء بدلاً من رمي خطأ.

```almarjaa
متجر قيمة = كائن؟.خاصية؛      # لا_شيء إذا الكائن فارغ
متجر عنصر = قائمة؟[فهرس]؛     # لا_شيء إذا الفهرس خارج النطاق
```

## 5. الجمل الشرطية والحلقات

### 5.1 الجمل الشرطية

#### شرط بسيط (If)
تُستخدم لتنفيذ كود معين إذا تحقق شرط محدد.

```almarjaa
إذا الدرجة >= ٩٠ {
    اطبع("ممتاز!")؛
}
```

#### شرط مع فرع آخر (If-Else)
يُنفذ كود بديل إذا لم يتحقق الشرط.

```almarjaa
إذا الدرجة >= ٦٠ {
    اطبع("ناجح")؛
} وإلا {
    اطبع("راسب")؛
}
```

#### شرط متعدد الفروع (If-ElseIf-Else)
يتحقق من عدة شروط بالتسلسل.

```almarjaa
إذا الدرجة >= ٩٠ {
    اطبع("ممتاز!")؛
} وإذا الدرجة >= ٨٠ {
    اطبع("جيد جداً")؛
} وإذا الدرجة >= ٧٠ {
    اطبع("جيد")؛
} وإلا {
    اطبع("مقبول أو راسب")؛
}
```

### 5.2 الحلقات

#### حلقة طالما (While)
تُكرر الكود طالما الشرط صحيح.

```almarjaa
متغير عداد = ١؛
طالما عداد <= ٥ {
    اطبع(عداد)؛
    عداد = عداد + ١؛
}
```

#### حلقة لكل (For)
تُكرر على عناصر مجموعة.

```almarjaa
لكل رقم في [١، ٢، ٣، ٤، ٥] {
    اطبع(رقم)؛
}
```

#### حلقة النطاق (Range)
تُكرر على نطاق من الأرقام.

```almarjaa
لكل i في نطاق(١، ١٠) {
    اطبع(i)؛
}

# مع خطوة مخصصة
لكل i في نطاق(١، ١٠، خطوة: ٢) {
    اطبع(i)؛    # ١، ٣، ٥، ٧، ٩
}
```

#### حلقة كرر (Repeat)
تُكرر كود عدد محدد من المرات.

```almarjaa
كرر ٥ مرة {
    اطبع("مرحباً")؛
}
```

#### التحكم في الحلقات
استخدم "توقف" لكسر الحلقة و"أكمل" للانتقال للتكرار التالي.

```almarjaa
لكل رقم في [١، ٢، ٣، ٤، ٥، ٦، ٧، ٨، ٩، ١٠] {
    إذا رقم > ٥ {
        توقف؛
    }
    اطبع(رقم)؛
}
```

## 6. الدوال (Functions)

### 6.1 تعريف الدالة

تُعرّف الدوال باستخدام الكلمة المفتاحية "دالة" متبوعة بالاسم والمعاملات والجسم.

```almarjaa
دالة اسم_الدالة(معامل١، معامل٢) {
    # جسم الدالة
    أرجع نتيجة؛
}
```

### 6.2 القيم الافتراضية

يمكن تحديد قيم افتراضية للمعاملات.

```almarjaa
دالة تحية(الاسم = "ضيف") {
    أرجع "مرحباً " + الاسم + "!"؛
}

اطبع(تحية())؛           # "مرحباً ضيف!"
اطبع(تحية("أحمد"))؛     # "مرحباً أحمد!"
```

### 6.3 التعليقات التوضيحية للأنواع

يمكن إضافة تعليقات توضيحية للمعاملات ونوع الإرجاع.

```almarjaa
دالة مساحة(الطول: رقم، العرض: رقم) -> رقم {
    أرجع الطول * العرض؛
}
```

### 6.4 الدوال المجهولة (Lambda)

تُعرّف الدوال المجهولة بصيغة مختصرة.

```almarjaa
متغير مربع = (ن) => ن ^ ٢؛
متجر مجموع = (أ، ب) => أ + ب؛

# استخدام
اطبع(مربع(٥))؛     # ٢٥
```

### 6.5 المولدات (Generators)

تُستخدم "أعطِ" لإنشاء مولدات تُنتج قيماً متعددة.

```almarjaa
دالة أرقام_حتى(ن) {
    متغير i = ٠؛
    طالما i < ن {
        أعطِ i؛
        i = i + ١؛
    }
}

لكل رقم في أرقام_حتى(٥) {
    اطبع(رقم)؛    # ٠، ١، ٢، ٣، ٤
}
```

## 7. الميزات المتقدمة

### 7.1 القوائم الاستيعابية (List Comprehension)

تُنشئ قوائم جديدة من قوائم موجودة بصيغة مختصرة.

```almarjaa
# مضاعفات الأرقام
متغير مضاعفات = [ن * ٢ لكل ن في [١، ٢، ٣، ٤، ٥]]؛

# مع شرط
متغير زوجية = [ن لكل ن في نطاق(١، ١٠) إذا ن % ٢ == ٠]؛
```

### 7.2 القواميس الاستيعابية

```almarjaa
متغير أسماء = ["أحمد"، "سارة"، "محمد"]؛

```

### 7.3 التفكيك (Destructuring)

```almarjaa
# تفكيك قائمة
متغير [أ، ب، ج] = [١، ٢، ٣]؛

متجر {اسم، عمر} = شخص؛
```

### 7.4 معالجة الأخطاء

```almarjaa
حاول {

    متغير نتيجة = قسمة(١٠، ٠)؛

    اطبع("حدث خطأ:"، خطأ)؛

    اطبع("تم التنفيذ")؛
}
```

### 7.5 الزخارف (Decorators)

```almarjaa
دالة حساب(ن) {

}
```

### 7.6 مديرو السياق (Context Managers)

```almarjaa
مع ملف كـ و {
    اطبع(و.اقرأ())؛
}
```

## 8. الدوال المدمجة

توفر لغة المرجع مجموعة واسعة من الدوال المدمجة:

### دوال الإدخال والإخراج
- اطبع(قيمة) - لطباعة قيمة
- ادخل(رسالة) - لقراءة إدخال من المستخدم

### دوال التحويل
- نص(قيمة) - تحويل إلى نص
- رقم(نص) - تحويل إلى رقم
- منطقي(قيمة) - تحويل إلى منطقي

### دوال القوائم
- طول(قائمة) - طول القائمة
- مجموع(أرقام) - مجموع العناصر
- متوسط(أرقام) - المتوسط
- أدنى(أرقام) - أصغر قيمة
- أقصى(أرقام) - أكبر قيمة
- رتب(قائمة) - ترتيب القائمة
- اعكس(قائمة) - عكس القائمة
- فريد(قائمة) - إزالة المكررات
- أضف(قائمة، عنصر) - إضافة عنصر
- أزل(قائمة، فهرس) - إزالة عنصر
- ابحث(قائمة، عنصر) - البحث عن عنصر

### دوال النصوص
- طول(نص) - طول النص
- كبير(نص) - تحويل لحروف كبيرة
- صغير(نص) - تحويل لحروف صغيرة
- اقتطع(نص، بداية، نهاية) - اقتطاع جزء
- انقسم(نص، فاصل) - تقسيم النص
- اجمع(قائمة، فاصل) - دمج النصوص
- استبدل(نص، قديم، جديد) - استبدال
- يبدأ_ب(نص، بداية) - التحقق من البداية
- ينتهي_ب(نص، نهاية) - التحقق من النهاية

### دوال القواميس
- مفاتيح(قاموس) - الحصول على المفاتيح
- قيم(قاموس) - الحصول على القيم

### دوال عامة
- نوع(قيمة) - نوع القيمة
- نطاق(بداية، نهاية) - إنشاء نطاق

## 9. أمثلة شاملة

### 9.1 برنامج ترحيب

```almarjaa
# برنامج ترحيب بسيط
اطبع("مرحباً بك في لغة المرجع!")؛
متغير الاسم = "المستخدم"؛
اطبع("أهلاً " + الاسم)؛
```

### 9.2 حاسبة بسيطة

```almarjaa
دالة جمع(أ، ب) { أرجع أ + ب؛ }
دالة طرح(أ، ب) { أرجع أ - ب؛ }
دالة ضرب(أ، ب) { أرجع أ * ب؛ }
دالة قسمة(أ، ب) {
    إذا ب == ٠ {
        أرجع لا_شيء؛
    }
    أرجع أ / ب؛
}

اطبع(جمع(١٠، ٥))؛    # ١٥
اطبع(طرح(١٠، ٥))؛    # ٥
اطبع(ضرب(١٠، ٥))؛    # ٥٠
اطبع(قسمة(١٠، ٥))؛    # ٢
```

### 9.3 فقاعة الفرز

```almarjaa
دالة فقاعة_فرز(قائمة) {
    متغير ن = طول(قائمة)؛
    لكل i في نطاق(٠، ن) {
        لكل j في نطاق(٠، ن - i - ١) {
            إذا قائمة[j] > قائمة[j + ١] {
                متغير مؤقت = قائمة[j]؛
                قائمة[j] = قائمة[j + ١]؛
                قائمة[j + ١] = مؤقت؛
            }
        }
    }
    أرجع قائمة؛
}

متغير أرقام = [٥، ٣، ٨، ١، ٩]؛
متجر مرتبة = فقاعة_فرز(أرقام)؛
اطبع(مرتبة)؛    # [١، ٣، ٥، ٨، ٩]
```

### 9.4 فيبوناتشي

```almarjaa
دالة فيبوناتشي(ن) {
    إذا ن <= ١ {
        أرجع ن؛
    }
    أرجع فيبوناتشي(ن - ١) + فيبوناتشي(ن - ٢)؛
}

لكل i في نطاق(٠، ١٠) {
    اطبع("F(" + نص(i) + ") = " + نص(فيبوناتشي(i)))؛
}
```

## 10. الملخص

لغة المرجع هي لغة برمجة عربية حديثة تجمع بين سهولة الاستخدام والقوة والأداء. توفر للمعربين برمجة طبيعية باللغة العربية مع ميزات متقدمة تنافس اللغات العالمية. البنية الواضحة والكلمات المفتاحية العربية تجعلها مثالية للمبتدئين والمحترفين على حد سواء.
